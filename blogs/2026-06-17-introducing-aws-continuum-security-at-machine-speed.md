AWS Continuum for code vulnerabilities, now in gated preview, addresses vulnerability lifecycles through discovery, prioritization, validation, and remediation. It reasons over complete environments—including infrastructure, permissions, topology, code, and organizational context—to confirm real issues and drive resolution, using multiple frontier models and starting in learn mode with human oversight before potentially graduating to automated enforcement based on user-defined risk categories.
